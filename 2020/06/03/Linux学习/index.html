目录结构

Bin：全称binary，含义是二进制。该目录中存储的都是一些二进制文件，文件都是可以被运行的。

Dev：该目录中主要存放的是外接设备，例如盘、其他光盘等。在其中的外接设备是不能直接被使用的，需要挂载（类似windows下的分配盘符）。

Etc：该目录主要存储一些配置文件。

Home：表示除了root用户以外的其他用户的家目录，类似于windows下的User/ 用户目录

Proc：全称process，表示进程，目录中存储的是linux运行时候的进程。

root：root用户单独的家目录。

Sbin：superbinary 该目录也是存放可以被执行的二进制文件，但是必须是有super权限的用户才能执行。

Tem：系统运行时产生的临时文件

Usr：存放用户自己安装的软件。类似于windows下的program目录

Var：存放程序或系统日志文件的目录。

Mnt：当外接设备需要挂载的时候，就需要去挂载到mnt目录下。

终端命令

格式： #指令主体 [选项] [操作对象]

关机：shutdown -h now（正常关机）、halt（关闭内存）、init 0

参考网址 man.linuxde.net

基础指令

ls指令

​ 含义：ls（list）

​ 用法一： # ls

​ 含义：列出当前工作目录下的所有文件/文件夹的名称

​ 注意：ls列出的结果颜色说明，其中蓝色的名称表示文件夹，黑色的表示文件，绿色的其权限为拥有全部权限。

用法二： # ls 路径

​ 含义：列出指定路径目录下的所有文件/文件夹的名称

​ 用法三： # ls 选项 路径

​ 含义：列出指定路径下文件/文件夹的名称，并以指定的格式进行显示。

​ 常见的语法：

用法四： # ls -lh 路径

​ 含义：列出指定路径下的所有文件/文件夹的名称，以列表的形式并且在显示文档大小时候以可读性较高的形式来显示。

pwd指令

​ 用法： # pwd （print working directory）

​ 含义：打印当前工作目录。

cd命令

​ 用法： # cd

​ 作用：用于切换当前工作目录

​ 语法：# cd 路径

​ 补充：在Linux中有一个特殊的符号“~“，表示当前用户的家目录。切换的方式 #cd ~

mkdir

​ 指令：mkdir 创建目录

​ 语法1： #mkdir 路径 【路径，可以是文件夹名称也可以是包含名称的一个完整路径】

​ 语法2：#mkdir -p

​ 语法3：#mkdir 路径1 路径2 路径三 （一次性创建多个目录）

touch指令

指令：touch

作用：创建文件

语法：# touch 文件路径 【路径可以是直接的文件名也可以是路径】

cp指令

指令： cp 用来复制文件/文件夹道指定位置

语法： # cp 被复制的文档路径 文档被复制到的路径

注意：1.Linux再复制过程中是可以重新对新位置的文件进行重命名的，但是如果不需要，则建议保证前后名称一致。

​ 2.当使用cp命令进行文件夹复制的时候需要添加选项”-r“【表示递归复制】，否则会被忽略。

mv指令

指令： mv 移动文档到新的位置

语法 ： #mv 需要移动的文档路径 需要保存的位置路径

注意： 可以用来重命名

rm命令

指令： rm 用来删除文档

语法： #rm 选项 文档的路径

选项： -f -r

注意：1. 在删除的时候如果不想频繁的确认，则可以再指令中添加选项 -f 表示 force

2. 在删除目录的的时候递归删除  加-rf  也不用频繁确认。
   3. `*`号（通配符）linux*表示只要文件以linux凯欧，后续字符则不管

vim命令

指令： vim （是一种文本编辑器） 打开一个文件（可以存在，可以不存在）

语法： #vim 文件的路径

退出打开的文件：安shrift+： 输入q 即可

wq保存退出

输出重定向

一般命令输出都会显示在终端中，有时候需要将一些命令的执行结果保存到文件中，这时候就需要用到输出重定向。

1.覆盖输出，会覆盖原先的文件内容

2.追加输出，不会覆盖，在末尾加

语法： #正常执行的指令> / >> 文件的路径

注意：文件可以不存在，不存在则新建。

cat指令

作用1：cat 有直接打开文件的功能，不需要推退出。

​ 语法： #cat 文件的路径

作用2：cat可以对文件进行合并

​ 语法：#cat 带合并的文件路径1 带合并的文件路径2 … 文件路径n > 合并之后的文件的路径。

进阶指令

df指令

作用：查看磁盘的空间

语法： #df -h （-h是以较高可读性）

​ mounted on 表示挂载点

free指令

作用：查看内存使用情况

语法：#free -m （-m 表示以mb为单位）

swap：用于临时内存，当系统真实不够用时，临时使用磁盘空间来充当内存

head指令

作用：查看一个文件的前n行，如果不指定n，则默认显示前10行

语法1： #head -n 文件路径

查看手册 man head

tail指令

作用1：用于查看一个文件的后n行，n不指定，默认显示后10行。

语法： #tail -n 文件路径

作用2：通过tail指令查看文件的动态变化

语法： #tail -f 文件路径 （手动添加不生效，输出重定向可以） 退出按下q

该命令一般用于查看系统的日志

less命令

作用：查看文件，以较少的内容进行输出，按下辅助功能键查看更多

语法： #less 需要查看的文件路径

辅助动能：输入20，从第20行开始 、空格键、上下键

wc指令

作用：用于统计文件内容信息（包括行数、单词书、字节数）

语法： #wc -lwc 需要统计的文件路径

​ -l： 表示lines 行数

​ -w：表示words 单词数

​ -c ：表示字节数

date指令

作用：表示操作时间日期（读取、设置）

语法1：#date 显示当前时间 输出：2020年6月8日 星期一 22：17：06

语法2：#date +%F 等价于（#date “+%Y-%m-%d”）*大小写严格 输出格式：2020-06-08

语法3：#date ”+%F %T“（引号表示让F、T成为一个整体，单引号也可以）输出格式 ：2020-06-08 22：22：22

​ 等价于： #date “+%Y-%m-%d %H:%M:%S”

语法4：获取之前或者之后的某个时间（备份）。

​ 一天前 #date -d “ -1 day” “+%Y-%m-%d %H:%M:%S”

​ 一天后 #date -d “ +1 day” “+%Y-%m-%d %H:%M:%S”

​ 符号可选值 ： +表示之后 -表示之前

​ 单位可选值： day、year、month

cal指令

作用：用来操作日历

语法1：#cal 等价于 #cal -1 直接输出当前月份的日历

语法2：#cal -3 输出上个月、当前月份、下个月份的日历

语法3：#cal -y 年份 输出某一年份的日历

clear/ctrl +L 指令

作用：清楚终端中已经存在的命令和结果（翻到了新的一页，以前的信息在上面）

语法：clear 或者快捷键 ctrl+L

管道

管道符： |

作用：管道一般可以用于过滤、特殊用法、扩展处理。

语法：管道不能单独使用，必须配合之前讲道德一些指令来一起使用。

案例1：通过管道符查询出根目录下包含y字母的文档名称

​ #ls /|grep y

案例2：通过管道操作方法实现less的等价效果（用不到）

​ 格式： #cat 文件路径 | less

案例3：扩展处理 ：用学过的命令统计某个目录下的文档总个数

​ #ls /|wc -l

高级指令

hostname指令

作用：操作服务器的主机名（读取、设置）

语法1：#hostname 含义：表示输出完整的主机名

语法2：#hostname -f 含义：表示输出当前主机名中的FQDN（全限定域名）

id指令

作用：查看一个用户的一些基本信息（包含用户id，用户组id，附加组id…..）该指令如果不指定用户则默认当前用户。

语法1：#id 默认显示当前执行该命令的用户的基本信息

语法2：#id 用户名

验证上述信息是否正确

验证用户信息，通过文件/etc/passwd

验证用户组信息，通过文件/etc/group

whoami指令

作用：显示当前的用户名，一般用于shell脚本，方便记录日志

语法：whoami

ps -ef指令（重点）

ps指令： 用于查看服务器当前的进程信息，类似于任务管理器

选项含义：

​ -e：等价于“-A”，表示列出全部的进程

​ -f：显示全部的列（显示全字段）

案例：在ps结果中过滤出想要查看的进程状态

​ #ps -ef | grep 进程名
$$

$$

top指令

作用：查看服务器的进程占的资源

语法： 进入命令： #top （动态显示）

​ 退出命令： 按下q键

方便快捷键（进入命令之后，按下方便的快捷键）：

​ M：按照内存MEM从高到低进行降序排列；

​ P：按照CPU使用率从高到低进行降序排列；

​ 1：当服务器拥有多个CPU时候可以使用“1”来切换是否展示显示各个CPU的详细信息。

du -sh指令

作用：查看目录的真实大小

语法： #du -sh 目录路径

选项含义：

​ -s： summaries，只显示汇总的大小

​ -h：表示以高可读性的形式进行显示

find 指令

作用：用于查找文件（参数有55个之多）

语法：#find 路径范围 选项 选项的值

选项： -name 按照文档名称进行搜索（支持模糊搜索）

​ -type 按照文档类型进行搜索

​ 文档类型：“-”表示文件 使用find时候用f替换

​ “d”表示文件夹

service指令（重点）

作用：用于控制一些软件的服务启动/停止/重启

语法： #service 服务名 start/stop/restart

例如：启动apache，其服务名httpd

​ #service httpd start
