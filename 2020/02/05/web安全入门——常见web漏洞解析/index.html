常见的安全问题

  1. 暗链：

    隐藏在网站中的链接，一般为网游/医疗/博彩/色情，能提高网站排名。

  2. 钓鱼：盗取用户信息。

  3. 篡改：攻破网站修改网页。

  4. webshell：攻破后留后门。

xss

xss是什么

  • 全称：Cross SIte Script

  • 中文名称：跨站脚本

  • 盗取用户信息、钓鱼、制造蠕虫等。

  • 概念：黑客通过“HTML注入”篡改网页，插入了恶意脚本，当用户在浏览网页时，实现控制用户浏览器行为的一种攻击方式。

这里的钥匙即为Cookie

xss分类

  • 存储型

    黑客构造xss脚本，xss的内容存储在数据库中，用户或管理员浏览时触发。由后端erb应用程序输出。

  • 反射型

    访问携带XSS脚本的连接就会触发XSS，数据存储在URL中，由后端erb应用程序输出。

  • DOM型

    访问携带XSS脚本的连接就会触发XSS，数据存储在URL中，由前端Javascript来输出。DOM型XSS其实是一种特殊类型的反射型XSS，它是基于DOM文档对象模型的一种漏洞。

    在网站页面中有许多页面的元素，当页面到达浏览器时浏览器会为页面创建一个顶级的Document object文档对象，接着生成各个子文档对象，每个页面元素对应一个文档对象，每个文档对象包含属性、方法和事件。可以通过JS脚本对文档对象进行编辑从而修改页面的元素。也就是说，客户端的脚本程序可以通过DOM来动态修改页面内容，从客户端获取DOM中的数据并在本地执行。基于这个特性，就可以利用JS脚本来实现XSS漏洞的利用。
