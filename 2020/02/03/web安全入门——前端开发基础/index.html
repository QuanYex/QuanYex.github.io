发表于 2020-02-03 | 更新于: 2020-02-04 | 分类于 学习

HTML：

  1. html 内容 ————-css 样式 —————js 动态

  2. html ：超文本标记语言，是web页面的基础。

  3. html格式：以开始标签起始，闭合标签终止，闭合标签比开始标签多一个斜杠。

    ​

    html元素属性：元素额外的信息

    标签属性：name、id、class

    事件属性：onload、onerror、onclick……

  4. html和dom树形结构关系：

  5. html转化为dom树的原因：更直观的了解页面元素、通过js可以对html进行任意操作。dom是文档对象模型,提供给JS很多操作页面中元素的属性和方法，本质是连接web页面和编程语言。js+dom是访问和操作html文档的标准方法。

JavaScript

  1. 那里能运行js： （1） html的之间，然后在事件属性中调用，例如onclick。（2） 浏览器的控制台中。

  3. js修改html元素内容：

  5. JavaScript BOM：获取浏览器信息和操作浏览器行为。BOM是浏览器对象模型，是用来连接浏览器和编程语言的。

    三个常用讲稿函数：警告弹窗alert（）、确认弹窗confirm（）、提示弹窗prompt（）。

    这三个常用于简单的调试和信息展示，如xss漏洞的测试

    (window.)screen 获取浏览器屏幕信息

    (window.)location 获取/控制用户页面URL

    (window.)navigator 获取访问者浏览器信息

  6. 获取用户的cookie：

    cookie通常是服务器发放给用户客户端的一小段文本信息。常见的场景就是用户输入用户名和密码成功登陆网站后，网站会生成一个cookie给用户，当作用户凭证，相当于我们的钥匙。
